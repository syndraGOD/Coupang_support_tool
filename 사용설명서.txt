[ 설치방법(베타 버전 한정) ]
1. 크롬 새로운 탭을 열고 chrome://extensions/ 입력
2. 우측 상단의 개발자 모드 ON으로 변경
3. 좌측 상단의 '압축해제된 확장 프로그램을 로드합니다' 클릭
4. 다운받은 폴더를 로드

[ 단축키 등록방법 ]
0. 개발자모드 ON / chrome://extensions/ 접속
1. 좌측 상단 햄버거 아이콘 클릭
2. 단축키 클릭
3. 연필 모양 아이콘 클릭한 뒤 ALT + A 입력

[ 사용설명서 ]
1. 크롬 tab123에 (젠데스크, ahri, eoc)순으로 놓아야 합니다.
2. ahri에서 현재 상담 or 작업 중인 고객을 반드시 맨 우측에 배치.
3. 확장 프로그램 단축키를 alt+a로 해두면 매우 편하게 사용하실 수 있습니다.
4. eta1이 넘었으면 빨간색, 안넘었으면 초록색으로 시간이 표기됩니다.
5. 자동채우기-내부메모는 현재는 ob(고객, edp ,store)매크로 기본 양식만 채워줍니다.
6. 최상단 고객-이름+전화번호 는 인입(젠데스크)이름 + 인입 전화번호입니다.

해당 프로그램은 (주)효성 ITX와의 협의 이후에 제작되었으며, 저작권 소유는 개발자에게 있습니다.
